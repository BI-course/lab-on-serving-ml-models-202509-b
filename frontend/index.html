List of API Endpoints

Classifiers

  • Decision Tree Classifier → POST /api/v1/models/decision-tree-classifier/predictions (test page)
  • Naive Bayes → POST /api/v1/models/naive-bayes-classifier/predictions
  • KNN → POST /api/v1/models/knn-classifier/predictions
  • SVM → POST /api/v1/models/svm-classifier/predictions
  • Random Forest → POST /api/v1/models/random-forest-classifier/predictions

Regressor

  • Decision Tree Regressor → POST /api/v1/models/decision-tree-regressor/predictions (test page)

Recommender & Clustering

  • Association Rules Recommender → POST /api/v1/recommender/association-rules
